Сочетание клавиш
----------------

| Символ | Сочетание клавиш                        |
|--------|-----------------------------------------|
| ®      | compose_key o r                         |
| ©      | compose_key o c                         |
| ™      | compose_key t m                         |
| ¼      | compose_key 1 4                         |
| ½      | compose_key 1 2                         |
| €      | compose_key e =                         |
| ¥      | compose_key y =                         |
| ≠      | compose_key / =                         |
| °      | compose_key o o                         |
| «      | compose_key right_shift+< right_shift+< |
| »      | compose_key right_shift+> right_shift+> |
| ¿      | compose_key right_shift+? right_shift+? |
| ←      | compose_key left_shift+< -              |
| …      | compose_key . .                         |
